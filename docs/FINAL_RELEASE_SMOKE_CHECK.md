# Final Release Smoke Check

## 当前结论

截至 `2026-03-16`，当前版本已经清掉首页的结构性阻塞，适合进入受控上线或公开 beta。

如果要作为“完全商用级正式发布”，上线前仍应完成下面这份人工验收。

## 已完成的代码侧检查

- 首页头部只保留一组语言切换
- 首页控制区已删除隐藏的旧语言切换残留
- 首页、详情页、本地摘要接口可正常返回 `200`
- 中文职业名翻译链路已接通

## 上线前人工验收

### 1. 首页

- 打开 [home.html](/E:/Codex/home.html)，确认首屏正常渲染
- 中英文切换后，标题、导航、筛选项、说明文案同步切换
- 鼠标滚轮缩放、拖拽、点击职业节点都可正常工作
- 四象限悬停说明能随鼠标出现

### 2. 筛选与搜索

- `日期` 下拉可切换
- `地区` 下拉可切换
- `职业大类` 下拉完整显示
- `影响类型` 下拉完整显示
- 中文搜索职业名可命中结果
- 搜索 SOC 代码可命中结果

### 3. 详情页

- 点击任一职业节点能进入详情页
- 详情页打开 [occupation-view.html](/E:/Codex/occupation-view.html) 不报错
- 中英文切换正常
- 趋势、分项解释、证据区能正常显示

### 4. 异常态

- 断开数据源时，页面应明确报错，不应静默回退成演示数据
- 严格模式下不应显示 mock 数据
- 页面不应出现空白区、重叠控件或明显错位

### 5. 兼容性

- Windows Chrome 检查一次
- Windows Edge 检查一次
- 手机端或浏览器移动端模拟检查一次

## 建议的发布判定

满足下面条件，可以判定为“可正式发布”：

1. 上述人工验收全部通过
2. 生产环境已启用真实数据源
3. 生产环境已启用严格数据模式
4. 域名、HTTPS、反向代理配置完成

如果第 1 条还没做完，建议先按“公开 beta”发布，不要直接定义成最终正式版。
